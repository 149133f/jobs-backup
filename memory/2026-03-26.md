## Chrome 瀏覽器自動化突破（重大！）

### 問題
- Python CDP 無法主動開啟瀏覽器
- browser tool 截圖會超時

### 解決方案：Chrome + Xvfb + browser tool 合作

1. **安裝 Google Chrome for Linux**
   - `sudo dpkg -i google-chrome-stable`
   - `sudo apt-get install -f -y`（修補依賴）

2. **安裝 Xvfb（虛擬顯示器）**
   - 讓 Chrome 在無頭環境運行

3. **啟動腳本** `/home/feng/.openclaw/workspace/scripts/start-chrome.sh`
   ```bash
   Xvfb :99 -screen 0 1920x1080x24 &
   DISPLAY=:99 google-chrome-stable --headless=new --remote-debugging-port=9223 ...
   ```

4. **OpenClaw Chrome Profile**（已設定）
   - `profile: chrome`
   - `cdpUrl: http://127.0.0.1:9223`

### 關鍵技術發現

| 發現 | 說明 |
|------|------|
| **browser tool + Python CDP 合作** | browser tool 返回 targetId，Python 用 CDP 直接連接 |
| **直接用分頁 ID 連接** | `ws://127.0.0.1:9223/devtools/page/{id}` |
| **Page.navigate 導航** | Python 可控制已存在的分頁導航到新 URL |
| **點擊 class** | `.more-button` 是「顯示完整內容」按鈕 |
| **抓取貼文內容** | `document.querySelector('ytd-backstage-post-renderer')?.innerText` |

### Python CDP 抓取範例
```python
page_uri = "ws://127.0.0.1:9223/devtools/page/{targetId}"
async with websockets.connect(page_uri, ping_interval=None) as ws:
    await ws.send(json.dumps({"id": 1, "method": "Page.navigate",
        "params": {"url": "https://www.youtube.com/@simpletainvest5376/posts"}}))
    await asyncio.sleep(5)
    await ws.send(json.dumps({"id": 2, "method": "Runtime.evaluate",
        "params": {"expression": "document.querySelector('.more-button')?.click()", "returnByValue": True}}))
    await asyncio.sleep(2)
    await ws.send(json.dumps({"id": 3, "method": "Runtime.evaluate",
        "params": {"expression": "document.querySelector('ytd-backstage-post-renderer')?.innerText", "returnByValue": True}}))
```

## Windows OpenClaw (賈維斯) 救援記錄

### 問題
- Port 18790 無監聽
- 賈維斯進程不存在

### 救援方式
使用 PowerShell 啟動 Windows OpenClaw：

```powershell
# 方式1：使用 openclaw.cmd
Start-Process -FilePath 'C:\Users\Feng\AppData\Roaming\npm\openclaw.cmd' -ArgumentList 'gateway','start' -WindowStyle Hidden

# 方式2：直接使用 node.exe（成功！）
Start-Process -FilePath 'C:\Program Files\nodejs\node.exe' -ArgumentList 'C:\Users\Feng\AppData\Roaming\npm\node_modules\openclaw\openclaw.mjs','gateway' -WorkingDirectory 'C:\Users\Feng\.openclaw' -WindowStyle Hidden
```

### 驗證
```bash
curl http://127.0.0.1:18790/json
```
