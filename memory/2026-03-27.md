## 瀏覽器端口鐵律（2026-03-28 新增）

### 端口分配（與 Windows 分開）
| 系統 | Chrome CDP 端口 |
|------|----------------|
| **WSL (Jobs)** | 9224 |
| **Windows (Brave)** | 9222 |

### 設定位置
- **OpenClaw config**: `browser.profiles.chrome.cdpUrl = http://127.0.0.1:9224`
- **啟動腳本**: `~/.openclaw/workspace/scripts/start-chrome.sh`

### 原因
避免 WSL Chrome (9223) 與 Windows Brave (9222) 衝突

### 鐵律
**WSL OpenClaw Chrome CDP 必須使用 9224**
