### 關鍵技術發現

| 發現 | 說明 |
|------|------|
| **browser tool + Python CDP 合作** | browser tool 返回 targetId，Python 用 CDP 直接連接 |
| **直接用分頁 ID 連接** | `ws://127.0.0.1:9223/devtools/page/{id}` |
| **Page.navigate 導航** | Python 可控制已存在的分頁導航到新 URL |
| **點擊 class** | `.more-button` 是「顯示完整內容」按鈕 |

### Python CDP 抓取範例
```python
page_uri = "ws://127.0.0.1:9223/devtools/page/{targetId}"
async with websockets.connect(page_uri, ping_interval=None) as ws:
    await ws.send(json.dumps({"id": 1, "method": "Page.navigate",
        "params": {"url": "https://www.youtube.com/@simpletainvest5376/posts"}}))
    await asyncio.sleep(5)
    # 點擊展開
    await ws.send(json.dumps({"id": 2, "method": "Runtime.evaluate",
        "params": {"expression": "document.querySelector('.more-button')?.click()", "returnByValue": True}}))
    await asyncio.sleep(2)
    # 抓取第一筆貼文
    await ws.send(json.dumps({"id": 3, "method": "Runtime.evaluate",
        "params": {"expression": "document.querySelector('ytd-backstage-post-renderer')?.innerText", "returnByValue": True}}))
```

## 股泰驛站自動抓取技術突破
- 問題：YouTube 貼文需要點擊「顯示完整內容」才能看到週守/月守
- 瀏覽器工具有超時限制，無法直接點擊
- 解決方案：**Python WebSocket CDP** 直接發送指令繞過超時
- Python 需要的套件：websockets
- 成功取得股泰驛站 20260319 資料：
  - TU: 34893, TM: 34003, TD: 32962
  - 週守: **33735**, 月守: **33592**
  - 外內盤量: -7277, 漲跌量: -3240

### Python CDP 範例
```python
import asyncio
import json
import websockets

async def click_button():
    uri = "ws://127.0.0.1:9222/devtools/page/{targetId}"
    async with websockets.connect(uri) as ws:
        script = """
        (() => {
            const btns = Array.from(document.querySelectorAll('tp-yt-paper-button'));
            for (const btn of btns) {
                if (btn.textContent.includes('顯示完整內容')) {
                    btn.click();
                    return 'clicked';
                }
            }
            return 'not found';
        })()
        """
        await ws.send(json.dumps({
            "id": 1,
            "method": "Runtime.evaluate",
            "params": {"expression": script, "returnByValue": True}
        }))
        response = await ws.recv()
        print(response)
```
