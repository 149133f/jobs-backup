# Chrome 擴展設定步驟（回家後操作）

## 步驟 1：安裝擴展
```bash
openclaw browser extension install
```

## 步驟 2：取得擴展路徑
```bash
openclaw browser extension path
```

## 步驟 3：在 Chrome 載入擴展
1. 打開瀏覽器
2. 前往 `chrome://extensions`
3. 啟用「開發者模式」（右上角）
4. 點擊「載入未封裝的擴展」
5. 選擇上面指令輸出的路徑

## 步驟 4：設定擴展
打開擴展選項，設定：
- **Port**: `18792`
- **Gateway Token**: `00a45ff64c77b8a0d22c9eff137bbe03b73ccec2864436a8`（複製這個！）

## 步驟 5：測試
1. 打開任意網頁
2. 點擊擴展圖標（應該顯示 ON）
3. 然後我就可以控制那個分頁了！
