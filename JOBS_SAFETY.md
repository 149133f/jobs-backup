## 🚫 絕對禁止事項（掛機紅線）

### 1. 禁止執行全局 Config 變更
```bash
# ❌ 絕對不能做：
openclaw config apply ...
openclaw config reset ...

# ✅ 正確做法：
# - 永遠用 isolated session 測試
# - 先備份：openclaw config get > ~/backup.json
# - 只用 config patch 局部修改
```

### 2. 禁止執行系統級重啟
```bash
# ❌ 絕對不能做：
openclaw gateway restart
systemctl restart openclaw-gateway

# ✅ 正確做法：
# - 除非主人明確要求
# - 重啟前必須先彙報
```

### 3. 禁止刪除任何檔案
```bash
# ❌ 絕對不能做：
rm -rf ...
del /f ...
Remove-Item ...

# ✅ 正確做法：
# - 只能讀取和複製
# - 刪除前必須徵求主人同意
```

### 4. 禁止執行未經測試的指令
```bash
# ❌ 絕對不能做：
# 任何你不確定後果的命令

# ✅ 正確做法：
# - 先問小小豬
# - 在 isolated session 測試
# - 確認影響範圍
```

## 🔴 緊急狀況處理

### 如果系統開始不穩定
1. **立刻停止**任何正在執行的操作
2. **發訊息**給小小豬：「豬姐，系統有狀況，需要協助」
3. **不要自己亂改**，等小小豬來處理

### 如果不小心執行了危險指令
1. **立刻彙報**小小豬，不要隱瞞
2. **說明做了什麼**
3. **等待指示**
